# Aura

**Python modular AI/bot project with separated AI, bot, configuration, data, and application entry-point layers.**

Aura is a Python software project organized around a modular application structure. The repository includes separate folders for AI logic, bot logic, configuration, data, and a main Python entry point.

Although this repository is not a trading-system project, it is useful as part of a broader technical portfolio because it shows Python project organization, modular thinking, and automation-oriented development.

---

## Project Purpose

The purpose of Aura is to experiment with a structured Python application architecture for AI/bot-style workflows.

The project demonstrates:

- Python application structure
- AI-related module separation
- Bot logic separation
- Config-driven development
- Data folder organization
- Main entry-point execution

---

## Repository Structure

```text
.
├── ai/          # AI-related logic and modules
├── bot/         # Bot-specific logic
├── configs/     # Configuration files
├── data/        # Data files / local storage
├── main.py      # Application entry point
└── README.md
```

---

## Main Components

| Path | Purpose |
|---|---|
| `main.py` | Application entry point. |
| `ai/` | AI-related logic and components. |
| `bot/` | Bot workflow / automation layer. |
| `configs/` | Configuration layer. |
| `data/` | Data storage or runtime data folder. |
